# Alcance del MVP

## Actual

- Crear tareas.
- Editar tareas.
- Completar tareas.
- Archivar tareas.
- Seleccionar grupo manualmente.
- Seleccionar responsable.
- Agregar fecha opcional.

- Compartir una lista entre múltiples usuarios.
- Actualizar la lista entre dispositivos.
- Ordenar tareas mediante reglas.
- Modificar el color según la antigüedad.
- Personalizar el tema visual por dispositivo.
- Usar Supabase como base remota.
- Usar almacenamiento local durante el desarrollo.

## Fuera del MVP (por ahora)

- Inteligencia artificial. 
- Clasificación automática. 
- Detección de tareas duplicadas. 
- Recurrencias. 
- Subtareas estructuradas. 
- Dependencias.
- Proyectos.
- Notificaciones.
- Sincronización offline avanzada.
- Historial completo de eventos.
- Resolución avanzada de conflictos.

## Principios

1. Los datos compartidos viven en la nube.
2. Las preferencias visuales viven en cada dispositivo.
3. La lógica de negocio no depende del proveedor de base de datos.
4. Las tareas conservan una descripción breve y detalles opcionales.
5. Las reglas deterministas controlan agrupación, orden y presión temporal.
